# Premium Stock Tracker 📦

ระบบจัดการสต็อกสินค้าและบัญชีสำหรับร้านค้าแบบพรีเมียม (Premium Stock Tracker) เป็นเว็บแอปพลิเคชันที่สร้างขึ้นมาเพื่อช่วยดึงข้อมูล (Scraping) ตรวจสอบ และจัดการข้อมูลสต็อกสินค้าประเภทบัญชี/รหัสผ่านจากแพลตฟอร์มต่างๆ เช่น FinShop, PuckPick และอื่นๆ ได้อย่างเป็นระบบ

## ฟีเจอร์หลัก ✨

- **ระบบดึงข้อมูลอัตโนมัติ (Automated Scraping):** สามารถเชื่อมต่อกับแพลตฟอร์มแหล่งสินค้า (เช่น FinShop, PuckPick) โดยใช้ Session Cookie และทำการดึงสต็อกล่าสุดของสินค้าแต่ละรายการมาแสดงผล
- **จัดการหมวดหมู่สินค้า (Dynamic Filtering & Categories):** 
  - ระบบรองรับการตั้งค่าชื่อหมวดหมู่ (Categories), ชนิดสินค้า (Types), ความชัด (Qualities) และจำนวนวัน (Sub Category 3) ผ่านเมนู Manage Settings
  - สามารถแก้ไขและจัดเรียงลำดับของตัวกรองได้อย่างอิสระ 
- **จัดการสินค้าได้แบบรายตัว:** 
  - สามารถแก้ไขรายละเอียดของสินค้า เช่น ชื่อสินค้า หมวดหมู่ย่อย และกำหนดสถานะการแสดงผล
  - การแก้ไขรายชื่อตัวกรองจะซิงค์กับข้อมูลสินค้าทั้งระบบ
- **Dashboard แสดงผลข้อมูลเชิงลึก:** สรุปจำนวนสต็อกรวม จำนวนสินค้าที่แสดง/ซ่อน และมูลค่ารวมของสต็อก

## เทคโนโลยีที่ใช้งาน 🛠️

- **Frontend:** React, TypeScript, Tailwind CSS, Vite, Lucide React (Icons), Shadcn UI
- **Backend:** Node.js, tRPC, SQLite (ผ่าน Drizzle ORM)
- **Scraper:** เช็คสต็อกสินค้าผ่าน API HTTP Client แบบไม่ต้องใช้ Headless Browser

## วิธีการติดตั้งและการใช้งาน 🚀

1. โคลน (Clone) โปรเจคลงในเครื่อง:
   ```bash
   git clone https://github.com/tanakronwork-oss/premium-stock-tracker.git
   cd premium-stock-tracker
   ```
2. ติดตั้ง Dependencies ทั้งหมด:
   ```bash
   npm install
   ```
3. เริ่มต้นระบบ (Run Development Server):
   ```bash
   npm run dev:full
   ```
   ตัวโปรเจคจะรันทั้ง Backend API (พอร์ต 3001) และ Frontend Vite (พอร์ต 3000) ไปพร้อมๆ กัน
4. เข้าใช้งานได้ที่เว็บเบราว์เซอร์: `http://localhost:3000`

## การตั้งค่าแหล่งสินค้า (Source Settings) ⚙️

หากต้องการให้ระบบดึงข้อมูลสต็อกได้ จะต้องเข้าไปใส่ **Cookie** ของแต่ละแหล่งดึงข้อมูล:
1. กดที่ปุ่มกล่อง **Settings** (มุมขวาบนของ Dashboard)
2. นำ Cookie ของ Session จากเว็บต้นทางมาวางลงในช่องที่กำหนด
3. ระบบจะบันทึกและทำการตรวจสอบสต็อกอัตโนมัติ (ผ่าน Cron Job) ให้เป็นระยะ
